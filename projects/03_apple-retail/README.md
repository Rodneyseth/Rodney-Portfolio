# Apple Retail Sales Analysis

End-to-end retail analytics across 1M+ Apple sales transactions.

## Folder Structure

| Folder | Phase | Contents |
|---|---|---|
| `01_data/` | Raw Data | CSVs from Kaggle — sales, products, category, stores, warranty |
| `02_sql/` | Schema & Cleaning | PostgreSQL scripts for Supabase — schema, cleaning, analytical views |
| `03_python/` | EDA & Time Series | Exploratory analysis, seasonal decomposition, product lifecycle |
| `04_forecasting/` | Revenue Forecast | Prophet 12-month forecast per category |
| `05_warranty/` | Warranty Analytics | Claim rates, repair status, store hotspots |
| `06_outputs/` | Chart Outputs | PNGs generated by Python scripts |
| `07_findings/` | Report | Findings PDF |
| `08_logs/` | Logs | Pipeline run logs |

## Dataset

Source: [Kaggle — amangarg08/apple-retail-sales-dataset](https://www.kaggle.com/datasets/amangarg08/apple-retail-sales-dataset)

Tables: `sales` (1M+ rows) · `products` · `category` · `stores` · `warranty`

## Stack

- **Supabase** (PostgreSQL) — data warehouse
- **Python** — pandas, statsmodels, prophet, matplotlib, seaborn, sqlalchemy
- **Power BI** — connected via psqlODBC to Supabase views
- **Excel** — pivot summary dashboards
