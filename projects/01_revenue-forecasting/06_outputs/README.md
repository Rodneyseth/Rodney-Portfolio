# 06_outputs — Generated Charts & Exports

This folder contains all charts and exports generated by running `03_python/forecasting.py`.

## Files generated after running the Python script

| File                          | Description                                      |
|-------------------------------|--------------------------------------------------|
| `01_monthly_trend.png`        | Monthly revenue trend line chart (2014–2017)     |
| `02_category_sales.png`       | Revenue by product category bar chart            |
| `03_discount_profit.png`      | Scatter plot: Discount rate vs Profit per order  |
| `04_actual_vs_predicted.png`  | Actual vs Predicted revenue (test set, 6 months) |
| `05_feature_importance.png`   | XGBoost feature importance bar chart             |
| `06_prophet_forecast.png`     | Prophet 12-month forecast (optional)             |
| `model_summary.csv`           | Model comparison table (MAE, RMSE, MAPE)         |

## How to regenerate

```bash
cd ../03_python
python forecasting.py
```

All charts are saved here automatically.

## Usage

These images are used in:
- The project README.md (embed with `![](06_outputs/04_actual_vs_predicted.png)`)
- The Excel dashboard (Insert → Picture)
- Your portfolio website writeup
- GitHub README preview
